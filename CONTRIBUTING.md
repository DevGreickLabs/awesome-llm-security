# Contributing to awesome-llm-security

Thanks for your interest! This guide will help you add resources to the list.

## How to Submit

1. Fork this repository
2. Create a new branch: `git checkout -b add/tool-name`
3. Add your resource to the appropriate section in `README.md`
4. Follow the format below
5. Submit a PR using the provided template

## Resource Format

Add entries in alphabetical order within their section:

```markdown
| [Tool Name](https://homepage.url/) | Short description (max 120 chars) | 🟢/🟡/🔴 | 🆓/💎/💰 | [GitHub](https://github.com/org/repo) |
```

## Quality Standards

Before submitting, ensure the resource:

- [ ] Is related to LLM/GenAI security
- [ ] Has a working link
- [ ] Is actively maintained (commit in the last 12 months)
- [ ] Has documentation or a README
- [ ] Is not a duplicate of an existing entry
- [ ] Description is concise (under 120 characters)
- [ ] Added in alphabetical order
- [ ] Has correct pricing badge

## Code of Conduct

Be respectful. Constructive feedback is welcome; hostility is not.
